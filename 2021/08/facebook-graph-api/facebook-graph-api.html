1. Giới thiệu

1.1 API là cái gì?

Ở đây mình không nói cụ thể, mình sẽ giải thích ở lĩnh vực mình đang đề cập, đơn giản bạn sẽ hiểu rằng API ở đây sẽ là một nguồn thông tin bạn sẽ lấy để phục vụ cho việc của bạn.

Có rất nhiều các API public bạn có thể dễ ràng tìm kiếm như Covid19 API, Weather Forecast API, ...

Ví dụ một link dẫn đến API Weather Forecast sẽ như dưới đây, các bạn có thể dễ dàng truy vấn các phẩn tử rồi dùng vào các công việc cần thiết.

1.2 Facebook Graph API là gì?

Theo như trang Facebook Dev:

Facebook Graph API là một API được phát triển bởi Facebook để phục vụ cho việc lấy thông tin các thành viên trong một trang web.

Nhưng nói một cách đơn giản, ở bài viết này mình sẽ dùng và một số thứ cần thiết để có thể tự đăng bài viết, tự comment, tự react, tự gửi tin nhắn, ...

Các bạn có thể dùng để nhắc lịch học, dự báo thời tiết hay gửi các thông tin mà bạn cần mỗi ngày.

2. Những thứ cần chuẩn bị

2.1. Facebook Page

Bạn cần tạo một Page Facebook. Để làm gì à? Để có thể cho con bot của bạn vào và nó có thể nhắn tin cho bạn những lúc bạn cần :). Nghĩa là Page đó sẽ có thể tự động đăng bài, tự động gửi tin nhắn theo ý muốn của chúng ta.

2.2. Tham gia Facebook for Developers

Bạn cần đăng kí vào Facebook for Developers để có thể tự động tạo một app vào trang của bạn.

Cách thức rất đơn giản, đầu tiên chỉ cần vào Facebook for Developers đăng nhập tài khoản Facebook, rồi nhập Email, nhập mã các thực là xong rồi.
